# Library Management System

A full-stack Library Management System built using Java Spring Boot, Spring Security, Spring Data JPA, MySQL, and Thymeleaf.

## Features
- User authentication & authorization
- Admin and Member roles
- Book borrowing system
- Secure login system
- Book management
- Responsive UI

## Technologies Used
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf
- Maven

## Roles
### Admin
- Manage books
- Manage users
- View borrowing records

### Member
- Browse books
- Borrow books
- Return books

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Configure MySQL database
4. Run the Spring Boot application
